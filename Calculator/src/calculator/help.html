Calculator Help

Numbers

Press number keys to build your operands for your calculation. e.g. If you require the number 1899 simply press the following button sequence:

1, 8, 9, 9

You can also add decimal points and negate the value if needed using the "+/-" and "." buttons respectively.

Operators

There are various operators available to use in your calculations, the basic operators are + - / *. Aditional operators are as follows

sqrt
This will form the square root of the current operand
%
The percent button will work out the percentage of the previous operand and perform the required operation on it e.g. 90 + 10% will calculate 90 + 9 as 9 is 10% of 90
1/x
Devides the reciprocal value of 1 over the current operand

Memory

There are a few memory functions to store, recall, clear. You can also use M+ to add the current operand to the value currently stored in memory
